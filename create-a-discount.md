# Create a percentage or fixed-amount discount

A discount reduces the price of eligible products by a specified percentage or a fixed amount.

---

## Prerequisites

Ensure the following before you begin:

* Permissions to create and manage discounts  
* Published products available for sale  

---

## Create a discount

To create a discount:

1. Go to **Promotions** and select **Create discount**.
2. Enter a name for the discount.
3. Under **Type**, select **Percentage** or **Fixed amount**, and enter the discount value.
4. Select which products the discount applies to: **All products** or **Selected products or product groups**.
5. Set a minimum purchase amount or minimum quantity to require customers to meet a threshold before the discount applies.
6. Under **Customer eligibility**, select **All customers** or **Selected customers or customer segments**.
7. Under **Combinations**, if available, select which other discounts this discount can be combined with.
8. Set the start date and optional end date.
9. Select **Save**.

---

## Notes

* Dates are evaluated using the store’s configured time zone.
* Only products included in the discount contribute toward minimum purchase thresholds.
* When applied to product groups, all variations of those products receive the discount.
* Discount combination rules determine whether multiple active discounts can apply to the same products.
